Module for the 'fog' gem to support ArubaCloud IaaS Service

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'fog-arubacloud'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fog-arubacloud
